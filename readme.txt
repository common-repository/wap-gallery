== Description ==

Framed Gallery allows you to display your photos in a grid layout with faux-wood frames.

There are 20 styles to choose from. Each example is headed by the correct style to use if you want to show that frame style.

= Shortcode =

[wap_gallery style="" cols"" limit=""]

**style:** use the headings from the examples section
**cols:** 1, 2, 3 or 4
**limit:** numeric

= Examples =

Some examples can be found [here](https://plugins.wearepanda.co.uk/wordpress-plugins/framed-gallery/)

== Installation ==

1. Unzip the wap-framed-gallery archive into your website`s `plugins` folder.
1. Visit your plugins page in the backend and activate the plugin
